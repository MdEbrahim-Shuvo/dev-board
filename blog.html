Question-1: What are the different ways to select an element in the DOM?

In JavaScript, there are several ways to select an element in the DOM. The most common method is document.getElementById("id"), which selects an element based on its unique ID. Another approach is document.getElementsByClassName("class"), which returns a collection of elements that share the same class name. Similarly, document.getElementsByTagName("tag") retrieves all elements with a specific tag name. A more flexible and modern way is using document.querySelector("selector"), which selects the first matching element based on a CSS selector.

Question-2: What is the difference between innerHTML, InnerText, and textContent?

innerHTML: Retrieves or sets HTML content, including tags and child elements. It processes HTML, making it useful for rendering dynamic content but risky for security if handling user input. innerText: Gets or sets only visible text, ignoring hidden elements and extra spaces. It reflects the rendered text as seen by the user. textContent: Retrieves or sets all text, including hidden elements, without formatting or processing HTML, making it safer for raw text handling.

Question-2: What is event delegation in the DOM?

Event delegation is a technique in the DOM where a parent element listens for events on its child elements using event bubbling. Instead of attaching event listeners to multiple child elements, a single event listener is added to their common parent. When an event occurs on a child, it bubbles up to the parent, which then handles it.

Question-4: How do you create, add, and remove elements using JavaScript?

In JavaScript, you can create, add, and remove elements dynamically in the DOM. To create an element, use document.createElement("tag"), then set its content or attributes. Adding the element can be done using appendChild() or append() to place it inside a parent, or insertBefore() to position it before another element. To remove an element, use element.remove() or parent.removeChild(element). These methods allow efficient DOM manipulation for dynamic web content.
